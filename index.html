Infraestructura de Comunicación Serverless: Portfolio, Email Corporativo y Formulario Seguro
Diseño e implementación de una infraestructura de comunicación completa sobre servicios gestionados en la nube, sin servidor propio ni coste de infraestructura.

Publicación y control de versiones: portfolio web estático desplegado en GitHub Pages con dominio personalizado. Gestión de registros DNS para asociar el dominio propio al despliegue. Control de versiones con Git: ramas, commits y seguimiento de cambios aplicado a código, scripts y configuraciones.

Email corporativo sin servidor: Cloudflare Email Routing actúa como capa de recepción, interceptando los mensajes entrantes al dominio propio y redirigiéndolos a una cuenta central. Brevo se integra como relay SMTP para el envío saliente autenticado bajo el dominio personalizado, con SPF y DKIM configurados para garantizar entregabilidad y evitar spoofing.

Formulario de contacto serverless y blindado: endpoint en Cloudflare Workers que recibe las peticiones del formulario y las enruta via API de Brevo. Seguridad implementada en capas: CORS estricto por origen, verificación anti-bot con Cloudflare Turnstile (token criptográfico validado en servidor) y credenciales gestionadas como Secrets encriptados — ninguna clave expuesta en el código fuente público.
